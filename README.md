# 🦖 Projeto Game Maker DinoChrome

Este é um projeto de jogo inspirado no famoso *Dino Game* do navegador Chrome, criado usando **GameMaker Studio**.  
O objetivo é desenvolver um jogo divertido onde um dinossauro corre pelo cenário e o jogador precisa desviar de obstáculos para sobreviver o máximo possível.

---

## 🎯 Sobre o jogo

- Jogo desenvolvido com o **GameMaker Studio**.
- Inspiração no clássico jogo do dinossauro do navegador Chrome.  
- inclui gráficos e sons semelhantes ao do jogo original.
- Feito juntamente com a playlist **"Seu primeiro jogo"** do canal **NoNeClass**
